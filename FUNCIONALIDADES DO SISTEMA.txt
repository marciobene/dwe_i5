FUNCIONALIDADES DO SISTEMA RASPIOT:

- ACESSAR TELA INICIAL
- IR PARA O SITE OFICIAL DO RASPBERRY PI
- CADASTRAR NOVO USUARIO (OBRIGATÓRIO EMAIL E SENHA)
- LOGAR NO SISTEMA (SE TENTAR ENTRAR DIRETO SEM LOGAR SISTEMA VOLTAR PARA TELA INICIAL)
- ACESSAR INFORMAÇÕES DE SENSOR DE TEMPERATURA
- FAZER BUSCA DAS LEITURAS FEITAS PELO SENSOR
- FILTRAR BUSCA DAS LEITURAS FEITAS PELO SENSOR
- ATUALIZAR INFORMAÇÕES DE CADASTRO
- DELETAR CONTA
- ACESSAR LOG DE ACESSO AO SISTEMA (SOMENTE USUARUIO COM admin == true)
- SAIR DO SISTEMA
